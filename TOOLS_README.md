## 2. Tools Suited With an API

### 2a. Built tools that already have an API endpoint

These tools are implemented and call a backend route. Adding or documenting a public API would allow scripts, CI, and other apps to use them.

| Tool | API route(s) |
|------|----------------|
| QR Code Generator | `GET/POST /api/qr` |
| DNS Lookup | `GET /api/dns?name=...&type=A` |
| SSL Certificate Checker | `GET /api/ssl?url=...` |
| HTTP Headers Checker | `GET /api/http?url=...` |
| Redirect Checker | (can use `/api/http` for chain) |
| Meta Tag Analyzer | `GET /api/meta?url=...` |
| WHOIS Lookup | `GET /api/whois?domain=...` |
| Robots.txt Tester | `GET /api/robots?url=...` |
| cURL Tool | `POST /api/curl/request` |
| Image Compressor / Resizer / Cropper / Watermarker / Filters / Converter | `POST /api/image/process` |
| PDF Compressor | `POST /api/pdf/compress` |
| PDF Splitter | `POST /api/pdf/split` |
| PDF Organizer | `POST /api/pdf/organize` |
| HTML to PDF | `POST /api/html-to-pdf` |
| Markdown to PDF | `POST /api/markdown-to-html`, `POST /api/markdown-to-pdf` |
| Website Screenshot | `GET /api/website-screenshot?url=...` |
| Gradient Maker | `POST /api/gradient` |
| Square Unit Price Calculator | `GET /api/tools/square-unit-price-calculator?...` |
| Image to PDF | `POST /api/image/to-pdf` |
| PDF to Image | `POST /api/pdf/to-image` |

### 2b. Built tools that would benefit from adding an API

Currently client-only; exposing an API would help automation and scripting.

| Tool | Suggested API | Notes |
|------|----------------|--------|
| Password Generator | `GET /api/password/generate?length=16&...` | Scripts, CI, bulk generation |
| Hash Generator | `POST /api/hash` (text or file) | MD5/SHA* for pipelines |
| Base64 Encoder/Decoder | `POST /api/base64` (encode/decode) | Pipelines, automation |
| JSON Formatter & Validator | `POST /api/json/format`, `POST /api/json/validate` | CI, API consumers |
| Email Validator | `POST /api/email/validate` | Add deliverability (DNS/SMTP) server-side; UI already format-only |
| Redirect Checker | `GET /api/redirect-check?url=...` | Return chain + status for monitoring |
